# OpenLiving

轻量级桌面生活管理工具，常驻系统托盘，透明悬浮显示。

## 功能

- **待办事项** — 右键托盘 → 添加待办，弹出窗口输入；点击 ○ 或文字标记完成
- **日记灵感** — 右键托盘 → 日记灵感，打开独立面板写作、浏览、搜索
- **已完成计数** — 托盘图标显示已完成待办数量
- **拖拽** — 拖拽顶部空白区域任意移动
- **显示位置** — 右键托盘 → 显示位置，可选 左上/右上/左下/右下
- **开机自启** — 右键托盘 → 开机自启（带勾选）
- **快捷键** — `Esc` 隐藏，`Ctrl+Q` 退出

## 使用

直接双击 `openliving.exe` 即可运行。数据文件自动生成在同目录下。

## 文件

| 文件 | 说明 |
|------|------|
| `openliving.exe` | 主程序（Windows 单文件，无需 Python） |
| `todo.py` | Python 源码 |
| `data.json` | 待办数据（自动生成） |
| `diary.json` | 日记数据（自动生成） |
| `README.md` | 本文件 |
